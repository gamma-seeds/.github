# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability in any Synapticode repository, please report it responsibly.

Email: security@synapticode.ai
Subject: [SECURITY] Brief description

Do not open public issues for security vulnerabilities.

## Response Timeline

- Acknowledgement: within 48 hours
- Assessment: within 7 days
- Fix or mitigation: based on severity

## Scope

This policy covers all repositories under the github.com/synapticode organisation.

## Confidentiality

All Synapticode repositories except tern-sdk contain proprietary technology protected by 35 patents with 350+ claims. Access to private repositories requires a mutual NDA.
